Telegram Niren
=======


> version： v0.0.0

Use [Telethon](https://github.com/LonamiWebs/Telethon) to operate niren user.



## 使用方式


```
pipenv install

cp ./src/env.example.yml ./src/env.yml
# edit ./src/env.yml

pipenv run ./src/tgNiren.py
```
